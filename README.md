# SortingASMx86
** SORTING WITH x86 **

This project was done on Visual Studio 2019.

It is a x86 Assembly Laguage application, in which we have implmented the following sorting algorithm on ASM:
    
    1) Selection Sort
    
    2) Quick Sort
    
    3) Insertion Sort
    
    4) Comb Sort
    
    5) Bubble Sort
    
    6) Shell sort
    
    7) Heap Sort
    
    8) Merge Sort
